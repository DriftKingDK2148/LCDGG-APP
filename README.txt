LCDGG App is an application that provides a mobile version of https://lcdgg.thomascyrix.com/. (The Confederation of Big Geeks).

It provides quick access to the latest articles.

In addition, it will allow you a much more comfortable display, in full screen, and a little faster than if you were on a web browser.

This application will be updated every week with the 2 new articles.

For the moment, the new articles will arrive with the application update. You will have to ACTIVATE AUTOMATIC UPDATES ON F-DROID.
If you don't do it, you won't receive the new articles, nor the bug fixes.


As you will see, you will be able to contact the developer to report any bug, or even any suggestion for improvement!

This is an application released by a beginner developer, please take this into account when using it ^^.
